This is a project for holding the code for a bunch of DNS related 
tools for failingover from a primary instance to a secondary one.

Fod communicating with DNS servers a DNSJava library is used.

For SSH - JSch. This is used for remote execution of scrits or commands - mainly renaming files, 
whos existance is monitored by DNS http/ftp probes.
